# Vue Starter

This is a vue starter template. Ready for use with:
- Netlify forms
- Netlify CMS
- Google fonts
- Fontawesome
- Media queries plugin

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
